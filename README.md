## 주요 기술
- 이미지 크롤링 : 강아지 모델을 만들기 위한 필요한 이미지들을 대량으로 크롤링
- 객체 인식 : Dlib 라이브러리를 사용하여 강아지 얼굴을 인식  
- 품종 분석 : 객체 인식된 이미지를 사용하여 학습 시킨 모델로 강아지 품종을 분석
